Mobius Conversion Tracker allows you to place Google's conversion tracking code onto pages with an easy to use shortcode

== Description ==

Easily add conversion tracking to your shopping/product pages with our shortcode that supports all of Google's common attributes

== Installation ==

1. Download the plugin archive and expand it (you've likely already done this).
2. Put the 'Mobius Conversion Tracking' folder into your wp-content/plugins/ directory.
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' to enabled Mobius Conversion Tracker.
4. You can now use the shortcode!
5. Example Usage: [conversion_track id="123" language="en" format="123" color="ffffff" label="123ABC" value="123" remarket="FALSE"]
